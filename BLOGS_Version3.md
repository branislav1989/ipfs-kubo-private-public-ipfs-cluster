# IPFS & Decentralized Hosting — Curated reading list

Share these resources with customers who want to learn about IPFS, pinning, and hosting static sites or apps on the decentralized web. Each link includes a short note you can use in marketing, docs, or onboarding emails.

- Official docs — IPFS
  - https://docs.ipfs.io/
  - Practical reference and step-by-step guides (install Kubo, add files, gateways, CIDs, and developer how-tos).

- Official blog — IPFS
  - https://blog.ipfs.io/
  - News, deep dives, and feature announcements from the IPFS community.

- Protocol Labs (research & ecosystem)
  - https://protocol.ai/blog/
  - Broader research, ecosystem updates, and long-form posts about projects around IPFS & Filecoin.

- IPFS Cluster (pinning & coordinated pinning)
  - https://cluster.ipfs.io/
  - Docs and guides about ipfs-cluster — how to run coordinated pinning across nodes.

- Awesome IPFS (curated resources)
  - https://awesome.ipfs.io/
  - A community-curated list of tools, libraries, tutorials, and projects built on IPFS.

- Fleek Blog — hosting websites on IPFS
  - https://fleek.co/blog/
  - Tutorials and case studies about deploying and serving websites on IPFS with modern tooling and CI.

- Pinata Blog — pinning & developer workflows
  - https://blog.pinata.cloud/
  - Guides about pinning strategies, APIs, and how to manage content availability.

- Infura — IPFS product & docs
  - https://www.infura.io/products/ipfs
  - IPFS gateway & pinning product docs, and quickstarts for developers who need managed endpoints.

- Filecoin Blog — storage incentives & use cases
  - https://filecoin.io/blog/
  - Articles about long-term storage, integration with IPFS, and production use cases.

- Textile (Powergate / Buckets) — developer tooling
  - https://docs.textile.io/ and https://textile.io/blog/
  - Tools for app-level user storage and developer libraries that integrate with IPFS/Filecoin.

- freeCodeCamp / Tutorials (hands-on guides)
  - Example: How-to / tutorials on using IPFS for web apps (search freeCodeCamp for up-to-date step-by-step guides)
  - Good for customers who prefer long-form, beginner-friendly walkthroughs.

- DWeb Foundation — community & events
  - https://dweb.foundation/
  - Background on the decentralized web movement, projects, and community resources.
